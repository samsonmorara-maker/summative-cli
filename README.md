# Project Management CLI Tool
 A command-line application for managing users, projects, and tasks. 
 ## Installation 
 git clone git@github.com:samsonmorara-maker/summative-cli.git
 -- cd project-management-cli
 -- pip install -r requirements.txt
 
## Run Add User python cli.py add-user 
-- name "Alex" List Users python cli.py list-users Add Project python cli.py add-project 
-- user "Alex"
-- title "CLI Tool" Add Task python cli.py add-task 
-- project "CLI Tool" 
-- title "Implement CLI" Complete Task python cli.py complete-task 
-- project "CLI Tool" 
-- task "Implement CLI" View User Projects python cli.py user-projects 
-- user "Alex" 
Run to Tests
